Vim-toCterm by shawncplus

Script to add cterm support to color schemes without the GUIColorSchemes plugin

Dependencies

  • PHP >= 5
  • Vim >= 7

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/shawncplus/Vim-toCterm

The Vim.org page for this script can be found here

Usage

  1. Open the color scheme to convert in Vim
  2. Place 'xterm' in your executable path
  3. :so <path to Vim-toCterm>/tocterm.vim
  4. ???
  5. Profit
